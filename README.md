# 🚀 Space-Invaders Game

A simple arcade-style game built using Python and Pygame.

## 🎮 Features
- Player movement (left/right)
- Bullet firing system
- Enemy movement
- Collision detection
- Score system

## how to play
- Use Left/Right arrow keys to move
- Press Space to shoot 
- Destroy enemies and score points

## 🛠 Tech Used
- Python
- Pygame
